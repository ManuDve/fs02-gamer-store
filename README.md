# Instrucciones para Devs

* Crear las ramas a partir de este release
* Nomenclatura de la rama: `feature/nombre-de-feature`
* Colocar los commits en modo imperativo y en inglés, ejemplo: `git commit -m "add new dependencies"`
* Enviar sus PR's a la rama de Release

# Setup

* Instalar Node en su última versión estable, v20+
* Instalar las dependencias con `npm i`
* Ejecutar comando para inicializar el proyecto con `npm run dev`
* Referirse al archivo `package.json` para consultar por otros comandos de testing.
